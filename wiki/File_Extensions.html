File Extensions

Many programs by Clickteam use their own proprietary format. This page lists them and the program they are associated to.

Extension Program Purpose Compatibility

(Icon for TGF)

.gam
  • The Games Factory
  • Click & Create
  • Klik & Play
 Stores assets, frames and events for the game/application. All Multimedia Fusion products can import data to a newer format, but cannot export back to .gam. Protected GAM files cannot be opened. Samples and music (MIDI) can be imported from the load dialog box by selecting this file type.
.cca
  • Multimedia Fusion 1.0-1.5
 Stores assets, frames and events as well as binary data for the game/application. Multimedia Fusion 2 and Clickteam Fusion 2.5 can import data to a newer format, but cannot export back to .cca. Samples and music (MIDI) can be imported from the load dialog box by selecting this file type.
.lib
  • Multimedia Fusion 1.0-1.5
 Sources of assets and objects in the form of library that can be easily added to a MMF1 project.

You might have different colors depending on which version is default.

.mfa
  • Multimedia Fusion 2
  • Clickteam Fusion 2.5
 Stores assets, frames and events as well as binary data for the game/application. Files created in MMF2 can be loaded in Fusion 2.5, but not vice versa. Projects saved in this format will be able to be imported to the successor product, currently referred to as Fusion 3.
.ccn
  • Multimedia Fusion 2
  • Clickteam Fusion 2.5
 Stores the runtime used by sub-applications and the Vitalize! runtime.
.iit
  • Install Maker
  • Install Creator 2
 Project file that stores the settings for creating installers.
.uts
  • Patch Maker
 Project file that stores the settings for creating patch programs.
