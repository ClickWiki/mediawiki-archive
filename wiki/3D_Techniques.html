3D Techniques

With the exception of Jamagic‎, the Raycaster Object, and Firefly (3D for Fusion), Clickteam's products target 2D animation and creation. This page is dedicated to listing the possibilities of going beyond 2-dimension by simulating a 3D environment or using advanced graphical extensions.

Technique / Method Difficulty Description External Resources

200px

Isometric Quite Easy This is a form of 2D that simulates the 3D perspective.

200px

Perspective Quite Easy Allows you to transform a 2D image into a 3D room illusion. Sparckman's 3D Room

200px

Raycasting Fairly Easy A paid extension called the Raycaster Object allows you to render a 3D environment from a 2D map. By Clickteam. On ClickStore

Firefly Medium A brand new set of objects to power CF2.5 with the ability to leverage the event editor to build your own 3D games and apps. By Clickteam. Firefly 3D Page On ClickStore

OpenGL Difficult Render actual 3D images within Fusion using OpenGL. These extensions are not actively being worked on, and can be considered outdated. 3D Extension List

Frameworks

The following are 3D engines programmed within Fusion.

Framework Required Cost Description
P3D $11.99 Consisting of 1000 events. This framework allows users to create small to mid-sized 3D games and applications.
